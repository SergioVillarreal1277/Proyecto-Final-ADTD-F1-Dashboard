# Proyecto-Final-ADTD---F1-Dashboard
Proyecto de análisis de datos de Fórmula 1 (1950-2025) desarrollado en Databricks. Incluye pipeline ETL completo con PySpark y SparkSQL, segmentación de escuderías mediante K-means y visualización de KPIs de rendimiento y patrocinios. Contiene el artículo técnico, notebooks de ejecución y documentación del dashboard.
